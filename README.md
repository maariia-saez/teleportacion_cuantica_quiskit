# Análisis de la Teleportación Cuántica con Qiskit

Este repositorio contiene el código fuente utilizado en el trabajo académico **"Análisis Teórico y Experimental de la Teleportación Cuántica: Del Teorema de No-Clonación a la Internet Cuántica"** (2025)

## Contenido
El script 'teleportacion_qiskit.py' incluye tres simulaciones:
  1. **Protocolo ideal**: Verificación teórica con fidelidad del 100%.
  2. **Experimento Zeilinger (1997)**: Simulación de las limitaciones de la BSM con óptica lineal (éxito probabilístico).
  3. **Hardware Real**: Ejecución en procesadores cuánticos de IBM para medir el ruido.

## Requisitos:
- Python 3.x
- Qiskit 1.0+
- Qiskit Aer
- Cuenta de IBM Quantum (para la simulación 3)
